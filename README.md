## Instalación

1. Clona el repositorio:

    ```bash
    git clone https://github.com/mbentorupo/Equipo09.git
    ```

2. Ingresa al directorio del proyecto:

    ```bash
    cd equipo09
    ```

3. Instala las dependencias:

    ```bash
    npm install
    ```
